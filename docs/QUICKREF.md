# 快速参考指南

## 🎮 播放控制

| 功能 | 按钮 | 说明 |
|------|------|------|
| 播放/暂停 | ⏯️ 播放/暂停 | 切换播放状态 |
| 停止 | ⏹️ 停止并返回列表 | 停止播放并返回文件列表 |
| 快退 | ⏪ 后退 | 后退30秒 |
| 快进 | 前进 ⏩ | 前进30秒 |
| 增大音量 | 🔊 增大音量 | 音量增加10% |
| 减小音量 | 减小音量 🔉 | 音量减少10% |
| 静音 | 🔇 静音 | 切换静音状态 |
| 全屏 | 全屏 🖥️ | 切换全屏模式 |

## 📺 集数切换（新功能）

| 功能 | 按钮 | 说明 |
|------|------|------|
| 上一集 | ⬅️ 上一集 | 播放上一个视频 |
| 下一集 | 下一集 ➡️ | 播放下一个视频 |

### 使用技巧

#### 快速切换集数
```
1. 在播放控制菜单中点击 "下一集 ➡️"
2. 系统自动切换到下一个视频
3. 无需返回文件列表
```

### 边界控制
```
第一集点击"上一集" → 提示"已到达列表起始位置"
最后一集点击"下一集" → 提示"已到达列表结束位置"
```

## 📂 文件浏览

| 功能 | 操作 | 说明 |
|------|------|------|
| 浏览目录 | 点击目录项 | 进入子目录 |
| 返回上级 | ◀️ 返回上一级 | 返回上级目录 |
| 选择视频 | 点击视频文件 | 开始播放 |
| 翻页 | 🔄/🔁 按钮 | 浏览更多文件 |

## 🔄 新功能使用流程

### 场景1：连续观看多集电视剧
```
1. 选择包含电视剧的目录
2. 点击第01集开始播放
3. 看完后点击"下一集 ➡️"
4. 自动播放第02集
5. 重复直到看完
```

### 场景2：停止后继续观看
```
1. 播放视频时点击"⏹️ 停止并返回列表"
2. 系统返回文件列表
3. 可以浏览其他视频
4. 点击任意视频继续播放
```

## ⚠️ 注意事项

1. **上一集/下一集按钮**
   - 仅在目录有多个视频时显示
   - 支持边界控制（无法越过首尾）

2. **停止并返回列表**
   - 自动返回到之前的文件列表
   - 视频列表信息会被保留

3. **全屏切换**
   - 点击后自动激活VLC窗口到前台
   - 支持多种VLC版本

## 💡 高级技巧

### 批量播放
1. 进入视频目录
2. 浏览所有视频
3. 使用上一集/下一集快速切换
4. 无需频繁返回目录列表

### 快速定位
1. 看完一集后立即点击"下一集"
2. 系统自动停止当前视频并播放下一个
3. 无缝衔接观看体验
